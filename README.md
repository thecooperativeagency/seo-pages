## Workflow
1. Wrench builds page → saved to store folder
2. Luc reviews → pushes to GitHub
3. GitHub Pages auto-publishes preview URL
4. Lance sends preview URL to web team via Slack
5. Web team copies content into Dealer eProcess (DEP)
